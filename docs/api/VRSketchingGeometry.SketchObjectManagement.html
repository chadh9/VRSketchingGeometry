Namespace VRSketchingGeometry.SketchObjectManagement

Classes

LinearInterpolationLineSketchObject

A line sketch object with no smooth interpolation between control point.

LineSketchObject

Provides methods to interact with a line game object in the scene.

PatchSketchObject

Patch surface sketch object using a grid of control points.

RibbonSketchObject

Ribbon sketch object. Creates a flat ribbon shaped geometry along control points and rotations.

SelectableObject

Abstract class for a object that can be selected. SketchObject and SketchObjectGroup are derived from this class.

SketchObject

Base class for different kinds of SketchObjects.

SketchObjectGroup

Represents a group of objects that implement IGroupable. SketchObjectGroups can contain SketchObjects and other SketchObjectGroups.

SketchObjectSelection

Represents a selection of SketchObjects and SketchObjectGroups. When the selection is activated the game objects are parented to the selection game object and highlighted. When deactivating the object return to their original position in the hierarchy. There can only be one active selection at a time.

SketchWorld

The root of a sketch. This is used to serialize, deserialize and export sketches.

Interfaces

IBrushable

Classes that implement this interface can work with Brush objects.

IGroupable

Classes that implement this interface can be added to SketchObjectGroups.

IHighlightable

Classes that implement this interface provide methods for visually highlighting their geometry.

ISelectable

This interface combines IGroupable and IHighlightable.
